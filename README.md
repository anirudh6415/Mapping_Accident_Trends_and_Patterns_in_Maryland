# For Data Pre-Processing
1 - Run "npm install" which will install all node modules ( Node.Js is required on system )
2 - Then use command "npm run dev"
3 - Call the get api with endpoint "http://localhost:5000/data-cleaning" (Port is 5000 and hardcoded)
4 - Then final.csv will be ready
